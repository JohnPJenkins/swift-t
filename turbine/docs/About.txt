
= Turbine Guides

Currently served at: http://swift-lang.org/Swift-T/*.html

Edit sites.txt

Run ./build.sh to make HTML

(HTML is copied to turbine-*.html)

Preview in local browser

Commit txt files (to ExM repo)

Copy sites.html to turbine-sites.html

Copy turbine-*.html to https://svn.ci.uchicago.edu/svn/vdl2/www/Swift-T

Commit HTML (to VDL2 repo)

Install with push_to.sh or wait for auto-install
